# Prototype 

## Introdução 

Essa etapa é responsável por criar um protótipo do projeto. O protótipo é uma representação visual do projeto, que permite que os usuários interajam com o sistema e forneçam feedback.

## Metodologia

Foi desenvolvido um prototipo de alta fidelidade, utilizando a ferramenta Figma, com base no backlog definido na etapa de Decision.
