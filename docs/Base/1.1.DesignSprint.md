# 1.1. Módulo Design Sprint

A Design Sprint é uma metodologia criada pela Google Ventures que visa resolver problemas complexos em um curto período de tempo. 

## Unpack

O time analisa o problema, define o objetivo e reúne o máximo de contexto sobre o desafio. O objetivo dessa etapa é entender o problema e coletar a maior quantidade de insights para depois serem debatidos.

Os artefatos gerados nessa etapa se encontram no documento:

- [Unpack do Projeto](Unpack.md)

## Sketch

A equipe se reúne para gerar o máximo de soluções possiveis para o problema, com base no que foi definido na etapa anterior. Os artefatos gerados nessa etapa se encontram no documento:

- [Sketch do Projeto](Sketch.md)

## Decision 

Aqui a equipe decide quais soluções serão desenvolvidas, com base no que foi definido na etapa anterior. Os artefatos gerados nessa etapa se encontram no documento:

- [Decision do Projeto](Backlog.md)
